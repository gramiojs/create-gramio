# create-gramio

Scaffolding your [GramIO](https://gramio.netlify.app/) project with the environment with easy!

-   Npm

```bash
npm create gramio ./bot
```

-   Yarn

```bash
yarn create gramio ./bot
```

-   Pnpm

```bash
pnpm create gramio ./bot
```

-   Bun

```bash
bun create gramio ./bot
```
